# Datasets

Boole-ANN is evaluated on six public datasets. **All raw data is hosted by
third parties** -- this bundle does not redistribute it. Run
`bash scripts/02_download_datasets.sh` to fetch them (some require
manual registration).

| Dataset  | n        | dim | type  | predicates       | source                                                |
|----------|----------|----:|-------|------------------|-------------------------------------------------------|
| SIFT1M   | 1e6      | 128 | u8    | synthetic equality/range/DNF | http://corpus-texmex.irisa.fr/             |
| SIFT100M | 1e8      | 128 | u8    | synthetic equality            | https://big-ann-benchmarks.com/             |
| SIFT1B   | 1e9      | 128 | u8    | synthetic equality            | https://big-ann-benchmarks.com/             |
| GIST1M   | 1e6      | 960 | f32   | synthetic equality            | ftp://ftp.irisa.fr/local/texmex/corpus/       |
| DEEP10M  | ~10e6    |  96 | f32   | synthetic equality            | https://ann-benchmarks.com/                  |
| YFCC10M  | 1e7      | 192 | u8    | real Flickr tags (BigANN public 100K queries) | https://big-ann-benchmarks.com/ |
| LAION1M  | 1e6      | 512 | f32   | top-200 CLIP tags (CC license) | https://laion.ai/                          |

## YFCC10M (canonical workload)

Files expected under `data/raw/yfcc100m/`:

```
base.10M.u8bin                       # 10M x 192 vectors
query.public.100K.u8bin              # 100K queries
base.metadata.10M.spmat              # per-base tag CSR
query.metadata.public.100K.spmat     # per-query tag CSR
GT.public.ibin                       # 100K x 10 ground truth (filtered)
```

The conjunction ground truth for the 5 paper slices is generated by
`python/data_prep/gen_yfcc10m_gt_matched_v6.py`.

## SIFT100M / SIFT1B (scale-out)

Files expected under `data/raw/bigann/`:

```
bigann_base.bvecs       # 1B x 128 uint8
bigann_query.bvecs      # 10K queries
bigann_gnd/             # ground truth tarball
```

`python/data_prep/prepare_sift100m_unified.py` derives the 100M slice and
`sift100m_build_conjunction_data.py` generates the equality-inclusion overlay
following the BigANN predicate generator.

## LAION1M (independent dataset)

`python/data_prep/laion_prepare.py` reads the LAION-400M parquet shards,
extracts the top-200 CC-licensed tags, and produces a 1M-vector subset
with multi-tag rate ~77%.
